Linguagem R

Variáveis e funções internas


Na programação, variáveis são nomes que representam valores que podem ser manipulados durante a execução de um programa. Elas são elementos básicos, pois armazenam os dados que são necessários para o algoritmo.

Linguagem R

Estruturas de dados


As estruturas de dados servem para organizar, manipular e acessar dados em diversos formatos

Linguagem R

Criando funções


Em programação, uma função é um bloco de código que executa uma operação e que pode ser chamada por outros trechos do programa. As funções têm um nome, uma definição e podem ser invocadas, assim como as funções matemáticas.

Linguagem R

Data frame


Data frame é uma estrutura de dados que organiza dados em uma tabela bidimensional, semelhante a uma planilha ou a uma tabela de uma base de dados.

dplyr:: manipulação de dados


Média, desvio-padrão e N da idade, segundo sexo

titanic::titanic_train |>
    dplyr::filter(Survived == 0) |>
    dplyr::group_by(Sex) |>
    dplyr::summarise(
        `Média` = mean(Age, na.rm = TRUE),
        `Desvio-padrão` = sd(Age, na.rm = TRUE),
        N = dplyr::n()
    ) |>
    dplyr::mutate(
        across(c(2, 3), \(x) round(x, 1))
    )

Regressão logística e razão de chances


titanic <-
    titanic::titanic_train |>
    dplyr::mutate(
        Pclass = relevel(
            factor(
                Pclass,
                labels = c("1st", "2nd", "3rd")
            ),
            ref = "3rd"
        ),
        Survived = factor(
            Survived,
            labels = c("não", "sim")
        ),
        Sex = factor(Sex, levels = c("male", "female"))
    )

model <- glm(Survived ~ Sex + Pclass,
    family = binomial,
    data = titanic
)

    broom::tidy() |>
